# MigrationWizard

**Assistant de Migration de Profils Windows**

Version : 1.0.23.11

## Téléchargement

Téléchargez `MigrationWizard.exe` (43 MB) - Aucune installation requise !

## Utilisation

1. **Double-cliquer** sur `MigrationWizard.exe`
2. Choisir **Export** ou **Import**
3. Suivre l'assistant étape par étape

## Fonctionnalités

- Export/Import de profils utilisateurs complets
- Dossiers : Bureau, Documents, Images, Vidéos, Musique, Téléchargements
- Wi-Fi, Imprimantes, Lecteurs réseau
- Navigateurs : Chrome, Firefox, Edge (favoris, historique, extensions)
- Outlook (fichiers PST, signatures)
- Bureau : Fond d'écran, positions des icônes
- Barre des tâches et Menu Démarrer
